# Jawad Cash Management (JCM) — Complete Operational Manual
> **Secure, Resilient, Localized Cash Ledger with Dual-Gatekeeping and Firestore Sync**

Jawad Cash Management (JCM) is a commercial-grade Progressive Web Application (PWA) designed for precise daily tracking of cash allocations. Built to meet rigorous security and performance standards, the application runs entirely in the browser, functions with 100% autonomy offline, and synchronizes effortlessly with a cloud database when an active internet connection is detected.
